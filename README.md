# Portfolio Website

This repository contains the code for my personal portfolio website. The website showcases various projects I've worked on, along with my skills and experience. It's designed to be clean, responsive, and visually engaging.

You can view the live portfolio here: [pradheepportfolio.vercel.app](https://pradheepportfolio.vercel.app)

## Features

- **Dark/Light Mode Toggle:** Switch between light and dark themes.
- **Responsive Design:** Works seamlessly across different screen sizes.
- **Animated Transitions:** Smooth animations for better user experience.
- **Project Showcase:** Highlights key projects with interactive elements.
  
## Technologies Used

- **React** for frontend development
- **Framer Motion** for animations
- **GSAP** for scroll and element animations
- **Tailwind CSS** for styling
